# Excess Food Management System

## 📌 Description
This project is designed to reduce food waste by connecting restaurants with NGOs.

## 🚀 Features
- Restaurant can add excess food details
- NGOs can view and request food
- User authentication system
- Database connectivity

## 🛠 Technologies Used
- Java
- SQL (MySQL)

## 🎯 Purpose
To minimize food wastage and help needy people by efficient food distribution.

## 📂 Project Structure
- Main.java
- User.java
- FoodDetails.java
- DatabaseConnection.java
